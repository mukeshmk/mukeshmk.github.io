Image and Audio Captcha Breaker

mukeshmk/image-audio-captcha

Convolutional Neural Network Based Audio and Image Captcha Breaker.

The Image captcha's are generated using a python package captcha which rotates scales and translates the letters and also adds noise to the iamge making it difficult to read. The audio captcha is generated manually using about 30 different voices curated from various sources and randomly selecting a voice for each character in the captcha.

The captcha is then feed to a Convolutional Neural Network after various pre-processing steps, which then predicts what the captcha is with over 95% accuracy.

For further details please read the README.md file on my GitHub.

Connect 4 AI

mukeshmk/connect-4

A multi-agent implementation of the game Connect-4 using various algorithms like Monte Carlo Tree Search, Minimax and Exptimax algorithms. Apart from these algorithms/bots, simple bots like a random bot and a forward checking bot were also implemented.

The game is designed in such a way that it allows a Player vs Player mode, Player vs Agent and Agent vs Agent modes of games. An Agent's game play can be constomised by changing it's paramerters i.e., depth and max iterations, to change the difficult level of the game play.

For further details please read the README.md file on my GitHub.

Twitter Sentiment Analysis

mukeshmk/twitter-sentiment-analysis

Twitter Sentiment Analysis - An opinion poll where public tweets considered as an indicator to predict the actual election results. The set of gathered tweets was pre-processed before it was used for training the classifier. The following pre-processing steps were applied on the data-set: lower-case convertion, punctuation removal, stemming, stopwords removal, emoji convertion, case change, and application of word-digit boundaries.

These pre-processing steps were used to clean the tweets which was then feed into a Gaussian Naive Bayes Classifier, which was used to predecit the sentiment of the tweet and and then to obtain the results.

For further details please read the project report file on my GitHub.
